📋 MANUAL SETUP (Jika setup.bat tidak berjalan):

   1. Copy folder ke: C:\xampp\htdocs\Sate Taichan Lilit
   2. Buka XAMPP Control Panel
   3. Klik START → Apache (tunggu hijau)
   4. Klik START → MySQL (tunggu hijau)
   5. Buka: http://localhost/phpmyadmin
   6. Import file: db/database.sql
   7. Buka: http://localhost/Sate%20Taichan%20Lilit/
