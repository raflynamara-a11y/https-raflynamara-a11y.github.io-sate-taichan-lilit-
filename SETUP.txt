📋 LANGKAH SINGKAT SETUP:

1. NYALAKAN XAMPP
   ✓ Buka XAMPP Control Panel
   ✓ Start MySQL
   ✓ Start Apache

2. IMPORT DATABASE
   ✓ Buka http://localhost/phpmyadmin
   ✓ Klik SQL
   ✓ Copy-paste isi file: db/database.sql
   ✓ Klik Go

3. PINDAHKAN PROJECT KE HTDOCS
   ✓ Copy folder "Sate Taichan Lilit" ke C:\xampp\htdocs\
   ✓ Atau buat link di sana

4. BUKA BROWSER
   ✓ http://localhost/Sate%20Taichan%20Lilit/

SELESAI! ✓
